[PREUVE DE PROGRAMME EN LOGIQUE DE HOARE]


Lancer le script launch.sh dans le dossier racine pour compiler et executer le vérificateur ET l'interface graphique :
./launch.sh

Note : Nécessite d'installer sur la machine
- Bison et Flex
- Qt5

Note : Pour compiler et executer seulement l'analyseur ou l'interface,
se placer dans le dossier correspondant et lancer le script launch
./launch.sh

Deux options pour le launch de l'analyseur : 
    - sans fichier texte en argument : entrer le texte après l'execution du launch.
    - avec un fichier texte en argument (deux fichiers exemples sont déjà présents dans le dossier de l'analyseur : entree.txt et entree1.txt)

Note : Lors d'un launch de l'analyseur avec un fichier en argument, faire attention à sauter une ligne à la fin de la preuve pour qu'il y ait une ligne vide sous celle-ci.
       Seule la première ligne du fichier est vérifiée.
